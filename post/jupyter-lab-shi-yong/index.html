Jupyter Lab 快捷键参考

Jupyter Lab 使用

  • 启动：jupyter lab
  • edit mode： Enter
  • command mode ：Esc

command mode 中使用的快捷键

  • 运行选中的cell： Shift + Enter
  • 删除选中的cell： d，d
  • 在当前cell的下方新建cell： b
  • 在当前cell的上方新建cell：a
  • 合并cell：Shift + m
  • 当前cell内容格式切换：
    y-》code
    m-》markdown

其他快捷键参考jupyter notebook

home目录修改

修改Anaconda目录下的/etc/jupyter/jupyter_notebook_config.json
虚拟环境：envs/虚拟环境名称/etc/jupyter/

添加如下内容：
"notebook_dir":"要指向的目录"
